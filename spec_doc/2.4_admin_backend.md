# 2.4 HR 后台管理功能

## 功能概述

HR 后台模块为管理员提供统一的面试管理界面，支持登录认证、面试列表查看、详情展示、创建新面试和删除面试等完整功能。该模块是 HR 与系统交互的唯一入口。

## 使用场景

1. **查看面试列表**：HR 登录后查看所有面试记录，了解整体情况
2. **查看详情**：点击某场面试，查看候选人的完整问答和评分
3. **创建面试**：手动创建新面试，获取候选人链接
4. **删除面试**：删除测试或无效的面试记录

### 1. 登录认证

**代码位置**：[backend/app/api/admin.py:17-46](../backend/app/api/admin.py)

#### 核心逻辑
```python
@router.post("/login", response_model=Token)
def login(form_data: OAuth2PasswordRequestForm = Depends(), db: Session = Depends(get_db)):
    # 1. 查询管理员账号
    admin = db.query(AdminUser).filter(AdminUser.username == form_data.username).first()

    # 2. 如果账号不存在，检查是否为默认账号
    if not admin:
        if form_data.username == settings.ADMIN_USERNAME and form_data.password == settings.ADMIN_PASSWORD:
            # 创建默认管理员账号
            admin = AdminUser(
                username=settings.ADMIN_USERNAME,
                password_hash=get_password_hash(settings.ADMIN_PASSWORD)
            )
            db.add(admin)
            db.commit()
            db.refresh(admin)
        else:
            raise HTTPException(status_code=401, detail="Incorrect username or password")

    # 3. 验证密码
    if not verify_password(form_data.password, admin.password_hash):
        raise HTTPException(status_code=401, detail="Incorrect username or password")

    # 4. 生成 JWT Token
    access_token = create_access_token(data={"sub": admin.username})

    return {"access_token": access_token, "token_type": "bearer"}
```

#### JWT 实现

**代码位置**：[backend/app/services/auth.py](../backend/app/services/auth.py)

```python
from jose import jwt
from datetime import datetime, timedelta
from passlib.context import CryptContext

pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")

def verify_password(plain_password: str, hashed_password: str) -> bool:
    """验证密码"""
    return pwd_context.verify(plain_password, hashed_password)

def get_password_hash(password: str) -> str:
    """生成密码哈希"""
    return pwd_context.hash(password)

def create_access_token(data: dict) -> str:
    """生成 JWT Token"""
    to_encode = data.copy()
    expire = datetime.utcnow() + timedelta(minutes=settings.ACCESS_TOKEN_EXPIRE_MINUTES)
    to_encode.update({"exp": expire})

    encoded_jwt = jwt.encode(
        to_encode,
        settings.JWT_SECRET,
        algorithm=settings.ALGORITHM
    )
    return encoded_jwt
```

### JWT Token 结构

**Payload**：
```json

```

**字段说明**：
- `sub`：用户名（Subject）
- `exp`：过期时间（Unix 时间戳）

**Token 示例**：
```
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiJhZG1pbiIsImV4cCI6MTcwOTMwODgwMH0.abc123...
```

### Token 存储

前端将 Token 存储在 `localStorage` 中：
```typescript
// 保存
localStorage.setItem('admin_token', access_token);

// 读取
const token = localStorage.getItem('admin_token');
```

### 认证流程

```
1. 用户登录 → 后端验证账号密码 → 返回 JWT Token
2. 前端保存 Token 到 localStorage
3. 后续请求携带 Token：Authorization: Bearer {token}

```

## 安全性考虑

### 当前安全问题

1. **后台 API 未验证 Token**：任何人都可以访问 `/api/admin/interviews`
2. **默认密码过于简单**：`admin/admin123` 容易被猜测
3. **Token 存储在 localStorage**：存在 XSS 风险
4. **无 HTTPS**：Token 在传输过程中可能被截获
5. **无 CSRF 防护**：容易受到跨站请求伪造攻击

### 安全加固建议

#### 1. 实现 Token 验证

```python
from fastapi import Depends, HTTPException
from fastapi.security import OAuth2PasswordBearer
from jose import JWTError, jwt

oauth2_scheme = OAuth2PasswordBearer(tokenUrl="api/admin/login")

def get_current_user(token: str = Depends(oauth2_scheme)):
    try:
        payload = jwt.decode(token, settings.JWT_SECRET, algorithms=[settings.ALGORITHM])
        username: str = payload.get("sub")
        if username is None:
            raise HTTPException(status_code=401, detail="Invalid token")
        return username
    except JWTError:
        raise HTTPException(status_code=401, detail="Invalid token")
```

## 错误处理

### 常见错误场景

| 错误场景             | HTTP 状态码 | 处理方式                    |
|-------------------|-----------|---------------------------|
| 用户名或密码错误       | 401       | 显示"用户名或密码错误"         |
| Token 过期          | 401       | 跳转到登录页面              |
| 面试不存在           | 404       | 显示"未找到面试详情"          |
| 网络请求失败         | -         | 显示"请求失败，请重试"         |

### 前端错误处理

```typescript
// 全局 Axios 拦截器
axios.interceptors.response.use(
  response => response,
  error => {
    if (error.response?.status === 401) {
      // Token 过期，跳转登录
      localStorage.removeItem('admin_token');
      window.location.href = '/admin/login';
    }
    return Promise.reject(error);
  }
);
```

### 分页实现示例

```python
@router.get("/interviews")
def list_interviews(
    page: int = 1,
    page_size: int = 20,
    db: Session = Depends(get_db)
):
    offset = (page - 1) * page_size
    interviews = db.query(Interview)\
        .order_by(Interview.created_at.desc())\
        .offset(offset)\
        .limit(page_size)\
        .all()

    total = db.query(Interview).count()

    return {
        "data": interviews,
        "total": total,
        "page": page,
        "page_size": page_size
    }
```
