# 2.3 AI 评估功能

## 功能概述

AI 评估模块是系统的核心智能模块，负责三个关键功能：
1. **问题生成**：根据岗位和简历生成针对性面试问题
2. **语音转文字（STT）**：将候选人的语音回答转为文本
3. **智能评分**：分析候选人回答，生成评分和评语

该模块采用服务化设计，与面试流程解耦，便于独立测试和替换实现。

#### 生产实现（LLM 生成）

```python
import json
import openai
from ..config import settings

openai.api_key = settings.OPENAI_API_KEY

async def generate_questions(position: str = None, resume_brief: str = None) -> List[Dict]:
    """
    使用 LLM 生成面试问题
    """
    # 如果没有岗位和简历信息，返回默认问题
    if not position and not resume_brief:
        return DEFAULT_QUESTIONS

    # 构建 Prompt
    prompt = f"""
你是一名专业的 HR，负责为候选人生成面试问题。

候选人信息：
- 岗位：{position or "未知"}
- 简历摘要：{resume_brief or "未提供"}

请生成 3-5 个面试问题，要求：
1. 问题针对性强，考察候选人的经验、能力和潜力
2. 避免过于宽泛或模糊的问题（如"介绍你自己"）
3. 问题应循序渐进，从基础到深入
4. 如果岗位信息明确，问题应与岗位职责相关

请以 JSON 格式返回，格式如下：
[
  {{"order_index": 0, "question_text": "..."}},
  {{"order_index": 1, "question_text": "..."}},
  ...
]
"""

    try:
        response = await openai.ChatCompletion.acreate(
            model="gpt-4",
            messages=[
                {"role": "system", "content": "你是一名专业的 HR，擅长设计面试问题。"},
                {"role": "user", "content": prompt}
            ],
            temperature=0.7,
            max_tokens=500
        )

        # 解析返回的 JSON
        content = response.choices[0].message.content
        questions = json.loads(content)

        # 验证格式
        if not isinstance(questions, list):
            raise ValueError("Invalid format")

        return questions

    except Exception as e:
        # 出错时返回默认问题
        print(f"LLM 问题生成失败: {e}")
        return DEFAULT_QUESTIONS
```

#### 生产实现（Whisper API）

```python
import openai
from ..config import settings

openai.api_key = settings.OPENAI_API_KEY

async def transcribe_audio(audio_file_path: str) -> str:
    """
    使用 OpenAI Whisper API 转文字
    """
    try:
        with open(audio_file_path, "rb") as audio_file:
            response = await openai.Audio.atranscribe(
                model="whisper-1",
                file=audio_file,
                language="zh"  # 指定中文
            )

        transcript = response["text"]
        return transcript

    except Exception as e:
        # 出错时返回错误信息
        print(f"STT 转文字失败: {e}")
        return "[语音识别失败]"
```

**Whisper API 规格**：
- **模型**：`whisper-1`
- **支持格式**：mp3, mp4, mpeg, mpga, m4a, wav, webm
- **文件大小限制**：25 MB
- **支持语言**：中文、英文等 50+ 种语言

**示例输出**：
```
输入：音频文件（候选人说话）
输出："我叫张三，有3年的Python开发经验，主要使用FastAPI和Django框架..."
```

```python
import json
from aliyunsdkcore.client import AcsClient
from aliyunsdkcore.request import CommonRequest

def transcribe_audio_aliyun(audio_file_path: str) -> str:
    client = AcsClient(
        settings.ALIYUN_ACCESS_KEY_ID,
        settings.ALIYUN_ACCESS_KEY_SECRET,
        'cn-shanghai'
    )

    with open(audio_file_path, 'rb') as f:
        audio_data = f.read()

    request = CommonRequest()
    request.set_domain('nls-meta.cn-shanghai.aliyuncs.com')
    request.set_method('POST')
    request.set_version('2019-02-28')
    request.add_body_params('audioData', audio_data)

    response = client.do_action_with_exception(request)
    result = json.loads(response)
    return result['Result']
```

#### 生产实现（LLM 评分）

```python
import json
import openai
from ..config import settings

openai.api_key = settings.OPENAI_API_KEY

async def evaluate_interview(answers: List[Dict[str, Any]]) -> Dict[str, Any]:
    """
    使用 LLM 评估面试
    """
    # 构建 Prompt
    answers_text = "\n\n".join([
        f"问题 {a['question_index'] + 1}:\n回答: {a['transcript']}"
        for a in answers
    ])

    prompt = f"""
你是一名专业的 HR，负责评估候选人的面试表现。

候选人的面试回答如下：
{answers_text}

请根据以下维度评分（满分 100）：
1. communication（沟通表达能力）
2. experience（经验匹配度）
3. potential（发展潜力）

要求：
1. 给出总分（total_score，0-100）
2. 给出各维度分数（dimension_scores）
3. 给出 100-200 字的综合评语（comment）

请以 JSON 格式返回，格式如下：
{{
  "total_score": 85,
  "dimension_scores": {{
    "communication": 90,
    "experience": 80,
    "potential": 85
  }},
  "comment": "..."
}}
"""

    try:
        response = await openai.ChatCompletion.acreate(
            model="gpt-4",
            messages=[
                {"role": "system", "content": "你是一名专业的 HR，擅长评估候选人。"},
                {"role": "user", "content": prompt}
            ],
            temperature=0.5,
            max_tokens=800
        )

        # 解析返回的 JSON
        content = response.choices[0].message.content
        evaluation = json.loads(content)

        return evaluation

    except Exception as e:
        # 出错时返回占位评分
        print(f"LLM 评分失败: {e}")
        return {
            "total_score": 0,
            "dimension_scores": {},
            "comment": "[评分失败，请人工审核]"
        }
```

```python
    },
```

## 数据流向

### 问题生成流程
```
POST /api/interviews/create
  └─> generate_questions(position, resume_brief)
      ├─> 有岗位/简历信息？
      │   ├─> 是：调用 LLM 生成针对性问题
      │   └─> 否：返回默认问题
      └─> 返回问题列表 [{"order_index": 0, "question_text": "..."}, ...]
```

### STT 流程
```
POST /api/interviews/{token}/answer
  └─> 保存音频文件到本地
  └─> transcribe_audio(file_path)
      ├─> 调用 OpenAI Whisper API
      └─> 返回文字内容
  └─> 写入 answers.transcript
```

### 评分流程
```
POST /api/interviews/{token}/complete
  └─> 查询所有 answers
  └─> evaluate_interview(answers)
      ├─> 构建 Prompt（包含所有回答）
      ├─> 调用 LLM 生成评分
      └─> 返回 {"total_score": ..., "dimension_scores": {...}, "comment": "..."}
  └─> 更新 interviews.evaluation_result
```

## 成本估算

### OpenAI API 定价（2025 年）

| 服务          | 模型       | 定价                  |
|-------------|----------|---------------------|
| Whisper STT | whisper-1 | $0.006 / 分钟        |
| GPT-4       | gpt-4     | $0.03 / 1K tokens (输入)<br>$0.06 / 1K tokens (输出) |

### 单次面试成本估算

**假设**：
- 3 个问题，每题回答 1 分钟
- 问题生成：100 tokens (输入) + 200 tokens (输出)
- 评分：500 tokens (输入) + 300 tokens (输出)

**成本计算**：
```
STT: 3 分钟 × $0.006 = $0.018
问题生成: (100 × $0.03 + 200 × $0.06) / 1000 = $0.015
评分: (500 × $0.03 + 300 × $0.06) / 1000 = $0.033

总计: $0.066 / 次面试（约 ¥0.47）
```

### 成本优化建议
- 使用 GPT-3.5-turbo 替代 GPT-4（成本降低 90%）
- 本地部署 Whisper（免费，但需 GPU）
- 缓存问题生成结果（相同岗位复用）

## 错误处理

### 错误场景与对策

| 错误场景             | 处理策略                          |
|-------------------|----------------------------------|
| OpenAI API Key 无效 | 回退到占位实现（返回默认问题/文本）   |
| 网络超时            | 重试 3 次，失败后回退               |
| API 限流            | 返回错误提示，提示稍后重试            |
| LLM 返回格式错误     | 使用正则解析，失败后返回占位结果       |
| 音频文件损坏        | 返回 "[语音识别失败]"              |
